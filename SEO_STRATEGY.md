## 🏗️ 2. Technical SEO Architecture (The Backbone)

We are building a PWA (Progressive Web App), so technical SEO is critical to ensure Google sees "Pages" not just an "App".

### 2.1 The "Spider Web" Internal Linking Structure 🕸️
Every image is a door to another part of the site.
*   **Horizontal Linking:** "See more **Cyberpunk** prompts" (Category link).
*   **Vertical Linking:** "See more from **Midjourney**" (Tool link).
*   **Contextual Linking:** "Used in **UI Design**" (Use-case link).

### 2.2 Programmatic SEO Engine ⚙️
We will not write 10,000 pages manually. We will generate them.
*   **Formula:** `[Style] + [Subject] + [Tool] + Prompts`
*   **Examples:**
    *   `/prompts/cyberpunk-city-midjourney`
    *   `/prompts/minimalist-logo-dalle3`
    *   `/prompts/anime-character-stable-diffusion`
*   **Implementation:** Use Next.js/React Dynamic Routes to render these metadata-rich landing pages on the fly.

### 2.3 Performance & Core Web Vitals (Speed = Rank) ⚡
*   **LCP (Largest Contentful Paint):** < 1.2s. Use `AVIF` or `WebP` images served from Cloudflare R2.
*   **CLS (Cumulative Layout Shift):** 0. Reserve space for images before they load (Masonry Grid Skeleton).
*   **FID (First Input Delay):** Near 0. React 18 Concurrent features and efficient hydration.

## 🧠 3. "Semantic Search" Optimization (Google AI Overviews)

Google now answers questions directly. We want Arti Studio to provide those answers.

### 3.1 The "Prompt Recipe" Schema
We will invent a custom structure if we have to, but standard **`Recipe`** or **`HowTo`** schema works wonders here.
*   **Name:** "How to generate a Neon Cyberpunk City"
*   **Supply:** "Midjourney Subscription"
*   **Step 1:** "Copy this prompt..."
*   **Ste 2:** "Adjust --stylize parameter..."
*   **Result:** [Image Object]

### 3.2 NLP (Natural Language Processing) Optimization
Write descriptions that answer "How", "Why", and "What".
*   *Bad:* "A picture of a cat."
*   *Good:* "A hyper-realistic close-up of a Maine Coon cat with lighting inspired by Rembrandt, generated using --v 6.0 for maximum fur detail."
